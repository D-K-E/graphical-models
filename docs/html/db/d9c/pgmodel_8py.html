Detailed Description

Probabilistic Graphic Model with Factors

This file contains the main model that drives the inference algorithms. It extends the graph definition by adding a new set called a set of factors. The set of factors can be arbitrarily defined or can be deduced from edges using independence structure assumed by the model. The #PGModel is the most generic model, hence we do not assume a particular independence structure.
